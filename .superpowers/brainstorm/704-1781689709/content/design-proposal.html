資料模型 (Backend)

notes 資料表
id INTEGER PRIMARY KEY AUTOINCREMENT
title TEXT NOT NULL
content TEXT NOT NULL DEFAULT ''
created_at TEXT NOT NULL DEFAULT (datetime('now'))
updated_at TEXT NOT NULL DEFAULT (datetime('now'))

API 端點

GET /api/notes → 取得所有記事 (依 updated_at DESC)
POST /api/notes → 新增記事 { title }
PUT /api/notes/{id} → 更新記事 { title?, content? }
DELETE /api/notes/{id} → 刪除記事

前端元件

types/index.ts — 新增 Note, NoteCreate, NoteUpdate 型別
api/notes.ts — 封裝 Notes CRUD API 請求
components/
├── NoteInput.tsx — 輸入標題，按 Enter 或 ＋ 新增記事
├── NoteList.tsx — 記事列表（含空狀態、筆數統計）
└── NoteItem.tsx — 單則記事（點擊展開編輯標題+內容）
App.tsx — 在 TodoList 下方渲染 NoteList
