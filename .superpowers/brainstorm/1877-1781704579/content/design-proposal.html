📋 拖曳排序設計方案

使用 @dnd-kit 為 Todo List 和記事本加入拖曳排序

資料庫變更

todos 表新增
sort_order INTEGER NOT NULL DEFAULT 0

notes 表新增
sort_order INTEGER NOT NULL DEFAULT 0

查詢改為
ORDER BY sort_order ASC, id DESC

後端 API

PUT /api/todos/reorder
Body: { "ids": [3, 1, 2] }
將 ids 陣列中的每個 todo 依序設為 sort_order 0, 1, 2...

PUT /api/notes/reorder
Body: { "ids": [4, 3, 1] }
同上

前端元件變更

安裝 @dnd-kit
npm install @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities

TodoList.tsx
包裹 DndContext + SortableContext，每項用 useSortable

TodoItem.tsx
加入 drag handle 屬性 (整列都可拖曳)

NoteList.tsx
相同方式加入拖曳排序

NoteItem.tsx
加入 drag handle 屬性

App.tsx
拖曳完成後呼叫 reorder API

api/todos.ts & api/notes.ts
新增 reorderTodos() / reorderNotes() 函式

拖曳視覺效果

  • 拖曳時項目半透明 + 陰影
  • 拖曳經過時出現插入指示線
  • 放開後項目平滑移動到新位置
  • 自動儲存排序到資料庫
